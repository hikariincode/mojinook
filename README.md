# 🌸 Moji Nook

Moji Nook is a Japanese learning platform focused on helping beginners master Hiragana and Katakana through mnemonic learning, handwriting practice and quizzes. The goal is to create a calm, intuitive learning experience inspired by Japanese aesthetics.

## ✨ Features

- 📖 Learn Hiragana
- 📖 Learn Katakana
- ✍️ Handwriting Practice
- 🧠 Quiz Mode

## 🛠 Tech Stack

- Next.js 16
- React
- TypeScript
- Tailwind CSS

## 🚀 Getting Started

Clone the repository

```bash
git clone <repo-url>
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
